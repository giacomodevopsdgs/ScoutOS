# ScoutOS Vision

ScoutOS is a personal web intelligence platform for finding and evaluating useful opportunities online.

The project connects browser automation, structured extraction, local scoring, and AI-assisted analysis.

## Principles

- GitHub is the source of truth.
- ChatGPT is used for planning and reasoning.
- Codex is used for implementation.
- Documentation should evolve with the code.
- Prefer modular and maintainable components.
